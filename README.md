# Procurement Project Documentation

## Overview
The Procurement project is designed to manage the purchasing process within an organization. The system provides functionalities for requisition creation, approval workflows, purchase orders, inventory management, and vendor management.

## Features
- **Requisition Management**: Create and track purchase requisitions.
- **Approval Workflows**: Customize approval workflows based on company policies.
- **Purchase Orders**: Generate and manage purchase orders.
- **Inventory Management**: Track inventory levels and manage stock.
- **Vendor Management**: Maintain a database of vendors and their contact details.

## Installation
To install the Procurement project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Adrianlyle/Procurement.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Procurement
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```

## Technologies Used
- Node.js
- Express
- MongoDB
